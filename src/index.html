Inspecting Faces

This is a full-stack web application that leverages AI-driven facial recognition to identify faces in images. I integrated the Clarifai API for facial recognition, which initially presented challenges related to API response handling and latency. To resolve these, I optimized API calls and implemented caching strategies to enhance performance.
